Copilot 说

当然可以！下面是一个简单的Python请求示例，演示如何使用requests库发送一个GET请求，并处理响应：

python
import requests

# 发送GET请求
response = requests.get('https://jsonplaceholder.typicode.com/todos/1')

# 检查响应状态码
if response.status_code == 200:
    # 解析JSON响应
    todo_item = response.json()
    print(f"任务ID: {todo_item['id']}")
    print(f"任务标题: {todo_item['title']}")
    print(f"任务完成状态: {todo_item['completed']}")
else:
    print(f"请求失败，状态码: {response.status_code}")

这个示例代码使用requests库从一个示例API (jsonplaceholder.typicode.com) 获取一个任务的详细信息，并打印出来。你可以用这个模板来处理更多复杂的请求和响应。
