写一个短小的python request示例demo

好的，没问题！下面是一个使用 Python 的 requests 库发送 HTTP GET 请求的简单示例：

Python
import requests

# 发送 GET 请求
response = requests.get('https://api.example.com/data')

# 检查请求是否成功
if response.status_code == 200:
    # 获取响应数据
    data = response.json()
    print(data)
else:
    print('请求失败:', response.status_code)

代码解释：

  1. 导入 requests 库： 首先，我们需要导入 requests 库，这个库是 Python 中进行 HTTP 请求的利器。
  2. 发送 GET 请求： 使用 requests.get() 函数发送一个 GET 请求到指定的 URL。这里我们使用了一个示例 URL，你可以替换成你想要访问的 API 或网页的地址。
  3. 检查请求状态： 请求完成后，response 对象会包含响应的状态码。如果状态码为 200，表示请求成功。
  4. 获取响应数据： 如果请求成功，我们可以使用 response.json() 方法获取响应数据。这里假设响应数据是 JSON 格式的。
  5. 处理响应数据： 将获取到的数据打印出来，或者进行进一步的处理。

更多功能：

  • 发送 POST 请求：
    Python
    data = {'key1': 'value1', 'key2': 'value2'}
response = requests.post('https://api.example.com/data', data=data)
  • 设置请求头：
    Python
    headers = {'User-Agent': 'My User Agent'}
response = requests.get('https://api.example.com', headers=headers)
  • 添加参数：
    Python
    params = {'param1': 'value1', 'param2': 'value2'}
response = requests.get('https://api.example.com', params=params)
  • 处理其他响应格式： 除了 JSON 格式，还可以处理 XML、文本等其他格式的响应数据。

注意事项：

  • URL 替换： 请将示例中的 URL 替换成你实际要访问的 URL。
  • 数据类型： 确保发送的数据类型与 API 的要求一致。
  • 错误处理： 可以添加更多的错误处理逻辑，例如处理网络错误、超时等。
  • 安全： 如果涉及敏感数据，请注意保护好你的请求。
